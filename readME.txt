1.项目介绍：HavaAChat是一款即时通讯软件，实现微信核心功能，包括实时聊天、好友关系及后台管理等。
2.核心功能：
用户模块：支持登陆注册、Token鉴权、个人信息维护等，采用Redis感知用户在线状态，实现秒级下线感知。
聊天模块：支持单聊/群聊消息（文字/图片/文件）收发，基于Netty长连接实现消息实时推送，离线消息补推。
社交关系	：好友全生命周期管理（搜索/添加/删除/拉黑），群组维护（创建/邀请/踢除/解散）。
管理后台	：用户管理（更改/封禁/强制下线），群组管理（检索/更改/强制解散），支持版本更新与灰度发布功能。
3.项目设计：
安全体系：Interceptor路由拦截+AOP权限控制+JWT动态签名进行三重鉴权，拦截恶意接口请求。
缓存机制：使用Redis实现用户心跳检测、验证码时效控制、联系人列表缓存等功能，优化数据读取效率。
交互设计：对好友申请/删除/拉黑等用户交互业务设置多级校验机制，确保多个体交互行为严格符合业务逻辑。
